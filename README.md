✊✋✌️ Rock Paper Scissors Web App

An interactive Rock Paper Scissors game built using Python and Streamlit, where you can play against the computer and track your performance in real time.

---

🚀 Live Demo

👉 Play the game here: [Rock Paper Scissors](https://rockpaperscisortanu.streamlit.app/⁠)

---

🎮 Features

- 🤖 Play against the computer
- 🧠 Randomized computer moves
- 📊 Score tracking system (User vs Computer)
- 🔁 Rematch option to restart the game
- ⏳ Interactive UI with decision delay
- 📱 Responsive design for desktop and mobile

---

🛠️ Tech Stack

- Python
- Streamlit

---

🎯 How to Play

1. Choose Rock, Paper, or Scissors
2. Wait for the computer to make its move
3. The winner will be displayed after a short delay
4. Scores will update automatically
5. Click Rematch to play again
