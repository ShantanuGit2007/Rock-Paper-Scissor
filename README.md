# 📋 Rock Paper Scissors Game
A simple and interactive Rock Paper Scissors web application built using Streamlit.
This app allows users to play against the computer with real-time results and score tracking.
---

# 🚀 Features
-✊ Choose Rock, Paper, or Scissors
-🤖 Computer randomly selects its move
-🏆 Instant result (Win / Lose / Draw)
-📊 Score tracking system
-🔄 Option to play multiple rounds
-📱 Clean and responsive layout
